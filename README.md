# Applied AI Research · EPAM

Engineering research papers from EPAM Applied AI. Each report takes one frontier
or consumer-grade AI capability — memory, agents, gateways, observability — and
measures what it takes to make it enterprise-deployable.

**Live site: <https://agorokh.github.io/applied-ai-research/>**

Authored by Arseny Gorokh. Published independently. Working drafts.

## Format

Each report is a single self-contained HTML file in the EPAM Dark Mode
engineering subsystem aesthetic — Fraunces serif for display, JetBrains Mono
for labels and code, dark palette (cyan / purple / blue / mint on near-black).
Reports are designed to read at 1280px or wider.

## Contributing

The series invites contributions from EPAM Applied AI engineers. To propose a
report, open an issue on this repository with the abstract and the corpus you
plan to measure against.
